# StudyMate Agent

## 一、專案簡介

StudyMate Agent 是一個 AI 學習摘要與測驗生成系統。使用者可以將課程內容、課堂筆記或教材文字貼入系統，系統會透過 Gemini API 自動產生學習摘要、關鍵名詞、學習建議、考試重點與選擇題，協助學生進行課後複習與自我檢測。

本專案使用 Streamlit 建立簡易網頁介面，並結合生成式 AI 模型，使系統能夠根據使用者輸入的內容自動整理資訊與產生測驗題，達到 AI 輔助學習的目的。

---

## 二、專案動機

在學習過程中，學生常常需要花費大量時間整理課堂筆記、歸納重點與製作複習題。如果課程內容較多，學生可能會因為資訊量過大而難以快速掌握核心概念。

因此，本專案希望透過 Agentic AI 的開發概念，設計一個可以協助學生自動處理學習資料的 AI Agent。使用者只需要輸入課程內容，系統就能夠根據任務需求，自動完成摘要整理、重點萃取、測驗題生成與答案解析等工作。

---

## 三、系統功能

本系統目前具備以下功能：

1. 課程內容輸入
2. AI 學習摘要生成
3. 關鍵名詞整理
4. 學習建議生成
5. 考試可能重點整理
6. 測驗題自動生成
7. 題目數量選擇
8. 測驗難度選擇
9. 正確答案與解析生成
10. 一鍵清除輸入內容與結果


## 八、環境變數設定

本專案需要使用 Gemini API Key。

請在專案根目錄建立 `.env` 檔案，並加入以下內容：

```env
GEMINI_API_KEY=你的 Gemini API Key
```

注意：請不要將真正的 API Key 上傳到 GitHub。

## 十、系統操作流程

1. 使用者開啟 StudyMate Agent 網頁介面。
2. 將課程內容貼入文字輸入框。
3. 選擇測驗題數量。
4. 選擇測驗難度。
5. 點選「產生摘要」、「產生測驗題」或「全部產生」。
6. 系統透過 Gemini API 產生摘要與測驗內容。
7. 使用者可以根據 AI 生成結果進行複習與自我檢測。

---

## 十一、Agentic AI 概念說明

本專案具有 Agentic AI 的基本概念。系統並非只單純回覆使用者問題，而是根據使用者輸入的課程內容與指定任務，自動完成一系列學習輔助流程。

在本系統中，AI Agent 的任務流程如下：

1. 接收使用者輸入的課程內容。
2. 分析文字中的重要概念。
3. 整理學習摘要與關鍵名詞。
4. 根據內容產生測驗題。
5. 依照使用者選擇的題數與難度調整輸出。
6. 提供正確答案與解析。

透過這樣的流程，系統能夠扮演學習助教的角色，協助使用者更有效率地整理學習資料。

StudyMate Agent 是一個以生成式 AI 為核心的學習輔助系統。透過 Python、Streamlit 與 Gemini API，本專案完成了一個可以輸入課程內容、自動整理摘要並產生測驗題的 AI Agent。
